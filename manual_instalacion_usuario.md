## 1. ¿Qué es TallerTec?

TallerTec es el sistema oficial de gestión de talleres deportivos y culturales del Tecnológico Nacional de México Campus Matehuala. Permite a los estudiantes:

- Inscribirse en línea a los talleres disponibles sin necesidad de acudir presencialmente a una feria.
- Consultar en tiempo real las horas acumuladas en su taller.
- Solicitar la constancia de cumplimiento de las 20 horas requeridas para titulación.
- Recibir notificaciones sobre su progreso y el estado de sus trámites.

> **Importante:** TallerTec es una aplicación **web**. No es necesario descargar ni instalar ningún programa. Funciona directamente desde el navegador de internet de cualquier dispositivo (computadora, tablet o celular).

---

## 2. Requisitos del Sistema

Dado que TallerTec es una aplicación web, los requisitos son mínimos. Cualquier equipo con acceso a internet y un navegador moderno es suficiente.

### 2.1 Requisitos Mínimos de Hardware

| Componente           | Especificación Mínima                         |
|----------------------|------------------------------------------------|
| Procesador           | Intel Core i3 de 1ª generación o equivalente  |
| Memoria RAM          | 2 GB                                           |
| Almacenamiento       | No requiere espacio en disco                   |
| Conexión a Internet  | 3 Mbps (banda ancha o datos móviles 3G/4G)    |
| Pantalla             | Resolución mínima de 1024 × 600 píxeles        |
| Cámara               | Requerida únicamente para escanear códigos QR  |

### 2.2 Requisitos Recomendados de Hardware

Para una experiencia óptima, especialmente al usar el escáner de código QR:

| Componente           | Especificación Recomendada                     |
|----------------------|------------------------------------------------|
| Procesador           | Intel Core i5 / AMD Ryzen 5 o superior         |
| Memoria RAM          | 4 GB o más                                     |
| Conexión a Internet  | 10 Mbps o superior                             |
| Pantalla             | 1280 × 720 píxeles o superior                  |
| Cámara               | Webcam HD (720p) o cámara trasera de smartphone|

### 2.3 Sistema Operativo

El sistema es compatible con cualquier sistema operativo que ejecute un navegador moderno:

- Windows 7 / 8 / 10 / 11
- macOS 10.13 o superior
- Linux (cualquier distribución con navegador actualizado)
- Android 8.0 o superior (para uso desde smartphone)
- iOS 13 o superior (iPhone/iPad)

## 3. Compatibilidad de Navegadores

TallerTec funciona con los siguientes navegadores. Se recomienda tener siempre la versión más reciente instalada:

| Navegador              | Versión Mínima | Recomendado |
|------------------------|----------------|-------------|
| Google Chrome          | 88             | Sí          |
| Microsoft Edge         | 88             | Sí          |
| Mozilla Firefox        | 85             | Sí          |
| Safari (Mac / iPhone)  | 14             | Sí          |
| Opera                  | 74             | Sí          |
| Chrome para Android    | 88             | Sí          |

> **Nota:** Internet Explorer (cualquier versión) **no es compatible**. Si se usa este navegador, se recomienda instalar Google Chrome o Microsoft Edge.

### Cómo saber si mi navegador está actualizado

**En Google Chrome:**
1. Hacer clic en los tres puntos (⋮) en la esquina superior derecha.
2. Seleccionar **Ayuda** → **Información de Google Chrome**.
3. Chrome mostrará la versión actual y se actualizará automáticamente si hay una versión nueva disponible.

**En Mozilla Firefox:**
1. Hacer clic en las tres líneas (☰) en la esquina superior derecha.
2. Seleccionar **Ayuda** → **Acerca de Firefox**.
3. Firefox verificará e instalará actualizaciones automáticamente.

## 4. Acceso al Sistema

No es necesario instalar ningún programa. Para acceder a TallerTec, solo se necesita:

1. Abrir el navegador de internet (Chrome, Firefox, Edge, etc.).
2. En la barra de direcciones del navegador, escribir la dirección oficial del sistema:

```
https://tallertec.vercel.app
```

3. Presionar **Enter**.

Se mostrará la página principal de TallerTec con las opciones para iniciar sesión o registrarse.

> **Recomendación:** Guardar la dirección en los favoritos del navegador para acceder más rápidamente en el futuro. En Chrome o Edge: presionar `Ctrl + D`. En Firefox: presionar `Ctrl + D`.

## 5. Crear una Cuenta

El registro es necesario solo la primera vez. Si ya se tiene una cuenta, ir directamente al paso 7.

**Paso 1.** En la página principal, hacer clic en el botón **"Regístrate"** (esquina superior derecha).

**Paso 2.** Completar el formulario de registro con los siguientes datos:

| Campo             | Descripción                                               | Ejemplo                       |
|-------------------|-----------------------------------------------------------|-------------------------------|
| Nombre Completo   | Nombre y apellidos tal como aparecen en el expediente     | Juan Pérez García             |
| Número de Control | Número de control estudiantil de 8 dígitos                | 22ME0101                      |
| Correo Electrónico| Correo institucional o personal válido                    | juan.perez@ejemplo.com        |
| Contraseña        | Mínimo 8 caracteres. Combinar letras y números.           | Taller2026!                   |
| Confirmar Contraseña | Escribir la misma contraseña nuevamente               | Taller2026!                   |
| Carrera           | Seleccionar de la lista desplegable                       | Ingeniería en Sistemas        |
| Semestre          | Semestre actual                                           | 4                             |
| Teléfono          | Número de teléfono (opcional)                             | 4881234567                    |

**Paso 3.** Hacer clic en el botón **"Crear Cuenta"**.

**Paso 4.** Si los datos son correctos, se mostrará un mensaje de confirmación indicando que se ha enviado un correo de activación.

> **Importante:** El correo electrónico usado para el registro debe ser válido y accesible, ya que se enviará un enlace de activación que es obligatorio para activar la cuenta.

---

## 6. Confirmar el Correo Electrónico

Después de registrarse, el sistema envía automáticamente un correo de confirmación.

**Paso 1.** Abrir el correo electrónico registrado (revisar también la carpeta de **Spam** o **Correo no deseado** si no aparece en la bandeja de entrada).

**Paso 2.** Buscar un correo con el asunto **"Confirma tu correo electrónico"** o similar, enviado por TallerTec.

**Paso 3.** Hacer clic en el enlace o botón de confirmación dentro del correo.

**Paso 4.** El navegador abrirá TallerTec y la cuenta quedará activada automáticamente.

> **Si el correo no llega:** Esperar hasta 5 minutos. Si después de ese tiempo no llega, verificar la carpeta de Spam. Si persiste el problema, contactar al administrador del sistema a través de la sección **"Contacto"** en la página principal.

---

## 7. Iniciar Sesión

**Paso 1.** En la página principal de TallerTec, hacer clic en **"Iniciar Sesión"**.

**Paso 2.** Ingresar el correo electrónico y la contraseña registrados.

**Paso 3.** Hacer clic en **"Iniciar Sesión"**.

**Paso 4.** Si los datos son correctos, el sistema redirigirá automáticamente al panel personal del usuario (dashboard).

---

## 8. Primeros Pasos en el Sistema

Al iniciar sesión por primera vez, se mostrará el **Panel del Estudiante (Dashboard)**. Desde aquí se puede:

| Sección               | Descripción                                                    |
|-----------------------|----------------------------------------------------------------|
| Mi Código QR          | Código único personal que el encargado escanea para registrar asistencia |
| Mis Talleres          | Lista de talleres en los que el estudiante está inscrito       |
| Horas Acumuladas      | Progreso actual de horas completadas                           |
| Notificaciones        | Avisos del sistema sobre inscripciones, asistencias y constancias |
| Solicitar Constancia  | Acceso directo para solicitar la constancia de 20 horas        |

### Guardar el Código QR

El código QR personal es el método principal para que el encargado registre la asistencia. Se recomienda:

- Tomar una captura de pantalla del código QR y guardarla en el teléfono.
- Mostrarlo al encargado al inicio de cada sesión del taller.

---

## 9. Inscribirse a un Taller

**Paso 1.** En el dashboard, hacer clic en **"Explorar Talleres"** o ir al menú **"Talleres"**.

**Paso 2.** Se mostrará el catálogo con todos los talleres disponibles del período actual, organizados en:
- **Talleres Deportivos** (fútbol, voleibol, básquetbol, atletismo, ajedrez, entre otros)
- **Talleres Culturales**

**Paso 3.** Revisar la información de cada taller:
- Nombre del taller
- Horario y días de sesión
- Ubicación
- Cupo disponible
- Nombre del encargado responsable

**Paso 4.** Hacer clic en **"Inscribirme"** en el taller deseado.

**Paso 5.** Confirmar la inscripción en el cuadro de diálogo que aparece.

**Paso 6.** Si la inscripción es exitosa, el taller aparecerá en el dashboard con el estado **"Activa"**.

> **Restricciones:**
> - Solo es posible inscribirse si el período de inscripciones está abierto.
> - No es posible inscribirse si el taller no tiene cupo disponible.
> - Solo se puede estar inscrito en un taller por período académico.

---

## 10. Consultar Horas y Asistencias

**Paso 1.** En el dashboard, seleccionar el taller activo para ver el detalle.

**Paso 2.** Se mostrará un resumen con:
- Total de horas acumuladas
- Porcentaje de avance hacia las 20 horas requeridas
- Historial de asistencias registradas (fecha, hora de entrada, hora de salida, horas computadas)

**Paso 3.** El sistema actualiza las horas automáticamente cada vez que el encargado registra la asistencia, ya sea por escaneo de QR o de forma manual.

---

## 11. Solicitar una Constancia

Una vez que el estudiante haya completado las **20 horas** requeridas, puede solicitar su constancia de forma digital.

**Paso 1.** En el dashboard, hacer clic en **"Solicitar Constancia"** (esta opción aparece solo cuando se alcanzan las 20 horas).

**Paso 2.** Revisar los datos que aparecerán en la constancia:
- Nombre completo
- Número de control
- Nombre del taller
- Total de horas completadas
- Período académico

**Paso 3.** Hacer clic en **"Enviar Solicitud"**.

**Paso 4.** La solicitud quedará en estado **"Pendiente"** mientras la Oficina de Deportes la revisa y aprueba.

**Paso 5.** Cuando la constancia sea aprobada, el sistema enviará una notificación y el estado cambiará a **"Lista"** o **"Aprobada"**, desde donde se podrá descargar en formato PDF.

> **Tiempo de respuesta:** La Oficina de Deportes procesa las solicitudes en un plazo estimado de 1 a 3 días hábiles.

---

## 12. Recuperar Contraseña

Si se olvidó la contraseña de acceso:

**Paso 1.** En la página de inicio de sesión, hacer clic en **"¿Olvidaste tu contraseña?"**.

**Paso 2.** Ingresar el correo electrónico registrado en la cuenta.

**Paso 3.** Hacer clic en **"Enviar enlace de recuperación"**.

**Paso 4.** Revisar el correo electrónico (incluyendo carpeta de Spam). Llegará un mensaje con un enlace para restablecer la contraseña.

**Paso 5.** Hacer clic en el enlace del correo. El navegador abrirá una página para crear una nueva contraseña.

**Paso 6.** Ingresar la nueva contraseña (mínimo 8 caracteres), confirmarla y hacer clic en **"Restablecer contraseña"**.

**Paso 7.** Iniciar sesión con la nueva contraseña.

## 13. Preguntas Frecuentes

**¿Necesito descargar una aplicación?**
No. TallerTec es una aplicación web. Solo se necesita un navegador de internet actualizado.

---

**¿Puedo usarlo desde mi celular?**
Sí. TallerTec está diseñado para funcionar en cualquier dispositivo: computadora, tablet o smartphone. Sin embargo, para escanear el código QR de otro estudiante se requiere que el dispositivo tenga cámara.

---

**¿Por qué no me llega el correo de confirmación?**
Revisar la carpeta de Spam o Correo No Deseado. Si el correo no llega después de 10 minutos, puede deberse a un error al escribir el correo durante el registro. En ese caso, contactar al administrador a través del formulario de **Contacto** en la página principal.

---

**¿Puedo inscribirme a más de un taller?**
No. El sistema permite inscribirse a un solo taller por período académico, conforme al reglamento institucional.

---

**¿Qué hago si el encargado no registró mi asistencia?**
Comunicarse directamente con el encargado del taller para que registre la asistencia de forma manual desde su panel. Si persiste el problema, contactar a la Oficina de Deportes.

---

**¿El código QR cambia?**
No. El código QR es único y permanente para cada estudiante. Solo se genera una vez al crear la cuenta.

---

**¿Qué pasa si el sistema no carga?**
Verificar la conexión a internet. Intentar recargar la página presionando `F5` o `Ctrl + R`. Si el problema persiste, el sistema puede estar en mantenimiento. Intentar de nuevo más tarde.

---

**No recuerdo si ya completé las 20 horas. ¿Cómo lo verifico?**
Iniciar sesión y revisar el dashboard. El sistema muestra el total de horas acumuladas y el porcentaje de avance en tiempo real.
