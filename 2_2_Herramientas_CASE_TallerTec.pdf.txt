1. Introducción 
Las herramientas CASE (Computer-Aided Software Engineering) son el conjunto de aplicaciones, 
plataformas y servicios que el equipo Virgilio Power utiliza para diseñar, desarrollar, probar y desplegar 
TallerTec. Este documento cataloga cada herramienta con su propósito, configuración específica para 
el proyecto y las convenciones de uso acordadas por el equipo. 
La correcta configuración de estas herramientas desde el inicio del proyecto garantiza consistencia en 
el código, calidad continua y reproducibilidad del entorno de desarrollo en cualquier máquina del 
equipo.

3. Herramientas de Desarrollo 
3.1 Framework Principal — Next.js 14 
Atributo 
Detalle 
Versión 
Next.js 14.2.x con App Router 
Instalación 
npx create-next-app@latest tallertec --typescript --tailwind --eslint --app

/app (rutas), /components, /lib, /hooks, /types, /public

5.3 GitHub Actions — Pipeline CI 
Pipeline automático definido en .github/workflows/ci.yml: 
on: [push, pull_request] 
jobs: 
  ci: 
    runs-on: ubuntu-latest 
    steps: 
      - uses: actions/checkout@v4 
      - uses: actions/setup-node@v4  (node 20) 
      - run: npm ci 
      - run: npm run lint           # ESLint 
      - run: npm run type-check      # TypeScript 
      - run: npm run test:coverage   # Jest (min 70%) 
      - run: npm run build           # Build Next.js
